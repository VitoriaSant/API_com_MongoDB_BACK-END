# 🚀 Gestão de Clientes API

Uma API REST simples e eficiente para o cadastro e gerenciamento de clientes.

---

## 🛠️ Tecnologias Utilizadas

* **Node.js**
* **Express**
* **Prisma**
* **TypeScript** 
* **MongoDB & Mongoose**

---

## ⚙️ Pré-requisitos

Antes de começar, você vai precisar ter instalado em sua máquina:
* [Node.js](https://nodejs.org/en/)
* [Git](https://git-scm.com/)
* Uma instância do [MongoDB](https://www.mongodb.com/) rodando localmente ou uma conta no MongoDB Atlas.

---

## 🚀 Como Executar o Projeto

### 1. Clonar o Repositório

### 2. Instalar as Dependências
npm install

### 3. Configurar as Variáveis de Ambiente
Crie um arquivo .env na raiz do projeto e adicione as suas credenciais. Existe um arquivo .env.example como referência

### 4. Executar a Aplicação
npm run dev

### 5. Exemplo de Corpo da Requisição
JSON
{
  "name": "Vitoria",
  "email": "Vitoria@email.com",
  "status": true 
}
